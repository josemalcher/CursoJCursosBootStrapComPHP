Using Classes de Visibilidade do Bootstrap, Eu criei alguns grids de exemplo abaixo para que você possa examinar no seu browser. Esta primeira grid está usando todos os 4 tamanhos de grade combinadas em uma única linha.

indica a exibição do'extra small'. Esta grade é sempre horizontal

lg indica a exibição do 'large grid'. O grid empilha horizontalmente < 1200px.

md indica a exibição do 'medium grid'. O grid empilha horizontalmente < 992px.

sm indica a exibição do 'small grid'. O grid empilha horizontalmente < 768px.

xs indica a exibição do 'extra small'. Esta grade é sempre horizontal.
